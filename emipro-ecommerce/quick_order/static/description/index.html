Website Quick Order

Make your customer's Bulk Shopping hunt on your website, a matter of mere minutes with Odoo Quick Order Module. It lets your Odoo Website customers add products in bulk to the custom-made shopping lists from their Odoo Website account.

The quick order feature displays products in list view for the customers to quickly add the items to the shopping lists in bulk. Desired products can also be searched and added to the shopping lists. The shopping lists can be added directly to the cart to order products in bulk.

Multiple shopping lists can be created and saved permanently by the customers in their Odoo website account and reordered directly whenever the need arises.

  • Features
    • The registered Odoo website Customers can order products in bulk using quick order menu.
    • The product catalog is displayed as a list view to quickly add the products to the bulk shopping list from the quick order list.
    • The customers can create multiple shopping lists and add products to them.
    • Bulk add the products to the shopping list and directly put the entire list in the cart.
    • The customers can reorder the entire shopping list with one click.
    • No need to scroll down to look for the products. The customers can simply search and add the products to the bulk shopping list.
    • Once a product is added to the bulk shopping list, it is removed from the product list in Quick order menu.
    • The admin can view all the orders and saved shopping lists in the Odoo backend by activating the developer mode.
  • What is the Quick Order Menu?
    • The Quick order menu lists all the products on the website in a list view to allow easy and fast navigation to the customers.
    • The customers can scroll to find the desired products or use the quick order search bar to look for the products.
    • If a product has variants, they show up when the product is selected.
    • Once a product is selected and added to the Quick order list, it is removed from the menu to allows faster navigation.
  • What is a Quick Order List?
    • The desired products can be first added to the quick order list.
    • The quantity of every product added can be specified before adding the shopping list from the quick order list.
    • The customers can review the list and then add it to the shopping list after finalizing to reduce errors.
  • Website View
    • Product will searched automatically in Product list while typing keywords.

    • All the variants of a product (if any) will be displayed in a pop-up where we can select variants of our choice in Quick Order list.

    • Create multiple shopping lists by simply adding the products from Quick Order list.

    • Add Quick Order list into already existesd Shopping list.

    • See available Shopping lists.

  • History
    • After submit an Quick Order, it is available in the Quick Order History, where you can review the ordered Products, billing and shipping information.
    • For open and past Quick Orders are located, Go to User Acount >> My Account.
    • View Quick Order History.

  • Backend View
    • Translatable and Configurable message to be displayed on website.

    • See all Quick Order and Shopping lists created by users only when enable the Developer Mode.
