Frequently Asked Questions

Which Odoo version is this theme compatible with?

This theme is compatible with Odoo V12 and V13.

What does ‘Free 5 Hours Customization’ include?

We offer 5 hours of free customization work with the purchase of our Theme Scita. There are no restrictions on the type of customization work but if the amount of work requires us to work more than 5 hours then you’ll have to pay standard charges for customization services.

On how many Odoo instances can this theme copy be used?

It can be used on only one Odoo instance. To use this theme in multiple instances, you need to purchase the theme for multiple times.

What are the support and update policies?

This theme includes 3 months of support period and unlimited update period.

I need to see a demo of this theme before purchase. How can I see that?

Of course, you can redirect to our live theme preview from here

Do you have guidelines for installing this theme?

Yes, we have a user guide which includes installation and configuration steps. Please contact our support team at support@appjetty.com

Do you install the theme for us?

Yes, we do install the theme for you. It’s quite easy. You just need to purchase the theme along with our installation services & we would be happy to assist you.

Can the theme support languages other than English?

Yes, we can. We can customize the theme to support the language(s) of your choice.

Can you personalize theme for me?

Yes, of course. You can share your requirements at support@appjetty.com and we’ll get back to you as soon as possible.

Is this theme responsive?

Yes, it is responsive.

Will I be able to add my own CSS to the theme?

Yes, you can create new modules and extend our theme CSS to add all the required changes in that module. Please do not alter the theme CSS directly as it may break the theme design.

Is this theme compatible with Odoo Community as well as Enterprise?

Yes, all of our themes are compatible with Odoo Community as well as Odoo Enterprise (on-premise & Odoo.sh).

Is this theme compatible with Odoo's Multi Website Feature?

Yes, our theme scita is compatible with Odoo’s Multi Website Feature. You can add our theme scita for multiple websites if they all are on Odoo v12 but all the websites should be hosted on a single Odoo instance. To apply our theme for multiple websites, you need to have a technical understanding of Odoo.

Can I resell your theme?

No, you can’t. Your purchase of the theme only allows your commercial use which means you don’t have the rights to resell our theme.
